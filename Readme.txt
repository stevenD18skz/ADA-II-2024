Proyecto de Interfaz Gráfica en Python
Descripción
Este proyecto implementa una interfaz gráfica de usuario (GUI) utilizando las bibliotecas tkinter y customtkinter. La interfaz permite realizar diversas acciones, como seleccionar archivos, mostrar mensajes emergentes, y más.

Requerimientos
Este proyecto tiene algunos requerimientos específicos para funcionar correctamente. Además de las bibliotecas estándar de Python, necesitarás instalar las siguientes librerías adicionales:

Librerías requeridas
Tkinter:

Tkinter es la biblioteca estándar para construir interfaces gráficas en Python.
Viene preinstalada con Python, por lo que no necesitas instalarla manualmente.
CustomTkinter:

CustomTkinter es una extensión de Tkinter que permite crear interfaces más modernas y con soporte para temas oscuros.
Para instalarla, ejecuta el siguiente comando:

pip install customtkinter


Cómo ejecutar el proyecto

1. Instalar dependencias: Ejecuta el siguiente comando para instalar las librerías necesarias (si no lo has hecho ya):
pip install customtkinter

2. Ejecutar el programa: Para iniciar la aplicación, simplemente abre el archivo main.py y ejecútalo

3. Interfaz gráfica: Al ejecutar el programa, se abrirá una ventana gráfica que te permitirá interactuar con las funcionalidades del sistema.
    3.1 para resolver un problema se debera presionar en la parte de "Cargar Archivo" y luego seleccionar el probelma en el navegador de archivos
    3.2 una vez cargado el archivo se podra ver su contenido al lado izquierdo, y se podra escoger entre los 3 tipos de algoritmos para solucionar el programa 
    3.3 una vez escogido el algoritmo, se mostrara la solucion de este en la parte derecha de la ventana y a la par se guardala la respuesta en un txt justo en la misma carpeta donde
        se encuentra el main.py
